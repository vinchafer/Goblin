v2 schliesst die drei Lücken:

Lücke 1 · Mehr-Hunk

Eine Datei = eine Karte; mehrere Änderungen = Hunks darin, je einzeln ✓/✕ oder die ganze Datei per Daumen. Keine Flut, kein Alles-oder-nichts.

Lücke 2 · Fehler-Morgen

Was landete / was nicht, Grund in Klartext, vier ruhige Auswege (reparieren, Gutes behalten, nochmal, zurückrollen). Heute komplett ungelöst.

Lücke 3 · Übergabe-Moment

Architekt (Serif, warm) und bau-goblin (Mono, grün) sehen verschieden aus; dazwischen eine sichtbare Zäsur — der Plan wechselt die Hände.

Parität bewiesen: alle 35 Fähigkeiten des heutigen Tabs haben ein Zuhause (PARITY.md). Nichts entfernt — drei Dinge besser platziert (Diff/Injektion → Feed-Karten; FABs → Publish+„⋯“), ein Ding neu (Fehler-Zustand).

Design-System unverändert: Manrope / Instrument Serif / JetBrains Mono · --ink/--surface/--brand-green/--brand-gold/--bone · Lucide · keine Emojis · kein Gold-auf-Hell.
