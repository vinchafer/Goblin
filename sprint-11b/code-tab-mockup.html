Auf dem Desktop ist der Review-Feed eine rechte Spalte neben dem Editor — dieselben Diff-Karten, hier geklickt statt gewischt. Der Faden (Architekt → Bau) und das Logbuch unten sind dieselben Konzepte wie mobil. Ein System, zwei Reichweiten: am Desktop editierst du und reviewst; am Telefon reviewst du nur — und genau das ist die Befreiung.

Warum das „I'd invest“ ist: nicht weil der Editor hübscher ist, sondern weil das Primitiv falsch war. Der mobile Code Tab hört auf, ein Editor zu sein, und wird zum Leitstand für einen Builder, der über Nacht läuft.
